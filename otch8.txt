Проверка записи информации в файл и чтения информации из файла
Для открытия файла используется системный вызов open()
Открытие для записи: fd=open("myfile", O_WRONLY | O_CREAT, 0666);
myfile - относительный путь к файлу, O_WRONLY | O_CREAT - флаги для записи и создания файла, если он отсутствует, 0666 - права доступа к файлу (чтение/запись для всех пользователей)
Открытие для чтения: fd=open("myfile", O_RDONLY)

Запись информации производится при помощи системного вызова write():
size=write(fd, string, 20);
Первый параметр - файловый дескриптор, определяющий средство связи, через которое пишется информация
Второй параметр - источник информации (символьный массив)
Третий параметр - количество байт, которое планируется записать
size - значение количества фактически записанных в файл байт
Чтение информации производится при помощи системного вызова read():
size = read(fd? string? 20);
fd - файловый дескриптор
string - приёмник информации, куда записываются байтовый поток
20 - количество байт, которые требуется прочитать
size - количество фактически прочитанных байт

Системный вызов close() закрывает файл с дескриптором fd
